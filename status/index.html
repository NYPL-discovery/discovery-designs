Discovery Website Status

This diagram shows the current state of patron-facing Discovery website. Visit the Discovery Project Status for the state of the overall architecture. Links are also provided to the respective component in various states.

  • Pages in the functional state are green. This means this page has been implemented and is usable by actual users. Note that this does not mean complete, but functional.
  • Pages in the design state are yellow. This means this page has been "mocked-up" visually using real data and design components, but not yet implemented.
  • Pages in the wireframe state are red. This means the content and functionality of this page has been defined and visually represented, but not yet designed.
